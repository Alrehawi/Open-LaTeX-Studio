About

Open LaTeX Studio is an open source LaTeX editor, that aims to support remote collaboration between users. Unlike other dedicated services that allow project sharing, there will be no need to purchase any plans in order to obtain the space or allow more people to work on a document, since users will be able to store their projects on any, free cloud storage solution (for example Dropbox, Google drive). The GUI of the editor is built using the NetBeans Platform framework, what makes the usage of the editor simple and intuitive.
